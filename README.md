## 项目概述

业务分析报告自动化系统是一个**企业级智能分析平台**，专为电商、零售等行业设计，提供从数据收集到智能洞察的全流程自动化解决方案。系统采用**渐进式增强架构**，支持零依赖运行到完整功能的多级部署模式。

### 🎯 设计理念

- **🚀 零依赖运行**：无需任何外部依赖即可正常工作
- **📈 渐进式增强**：根据依赖可用性自动升级功能
- **🧠 智能洞察**：AI驱动的数据分析和建议生成
- **🏪 零售专业**：专门针对零售行业的业务分析报告
- **🎨 美观可视化**：从ASCII图表到交互式仪表板
- **📱 多端支持**：Web界面、移动端适配、API接口

#### 🚀 **快速启动指南**

```bash
# 检查Python版本 (需要3.8+)
python --version

# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate     # Windows

# 安装依赖
pip install -r config/requirements_level2.txt
```

##### 2. 配置检查
```bash
# 验证配置文件
python -c "from src.config.settings import settings; print('配置验证通过')"

# 检查模板文件
ls src/templates/
```

##### 3. 启动服务器
```bash
# 方式1：使用启动脚本
python scripts/start_server.py

# 方式2：直接启动
python src/main.py

# 方式3：使用uvicorn
uvicorn src.main:app --host 0.0.0.0 --port 8000 --reload
```

##### 4. 验证启动
```bash
# 检查服务状态
curl http://localhost:8000/health

# 访问Web界面
open http://localhost:8000

# 查看API文档
open http://localhost:8000/docs
open http://localhost:8000/api-docs
```

| **端口占用** | 8000端口被占用 | 修改端口：`uvicorn src.main:app --port 8001` |
